Bunq Fix CSV

This tool will fix the CSV file from bunq. It fixes the number format and the column names, ready to import into Xero.

Your data is safe. All processing happens locally on your browser - we never upload your CSV to our server.
Bunq Fix CSV is a tool to fix CSV files from bunq, especially for importing into Xero. See the code on Github. Copyright © 2022 Jason Norwood Young.
